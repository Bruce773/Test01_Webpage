Lists

Lists can be very helpful when you're trying to display a group of information that be longs together. There are several different types of lists. The three that I use are called Unordered, Ordered, and Descriptive. This is an example of an unordered list:

<ul>
	  <li>Item #1</li>
	  <li>Item #2</li>
	  <li>Item #3</li>
	</ul>

This is an example of an ordered list:

<ol>
	  <li>Item #1</li>
	  <li>Item #2</li>
	  <li>Item #3</li>
	</ol>

This is an example of an descriptive list:

<dl>
	  <dt>Item #1</dt>
	  <dd>Description of Item #1</dd>
	  <dt>Item #2</dt>
	  <dd>Description of Item #2</dd>
	</dl>
