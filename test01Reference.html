HTML Tags

Here you'll a few tags that may come in handy sometime.

Tag or Element Description
<h1> </h1> The header tags help with formatting. They can vary from h1 to h6.
<p></p> The paragraph tags allow you to seperate sections of your page into different paragraphs.
<br> The break tag will create a space on the HTML document whenever
it is called. It is also one of the only tags without an end tag.
<a href=""></a> A lot of people will think of this tag a a link tag... it's actual name is the anchor tag. It is essential the HTML tag used to link to things like IDs, external webpages, and webpages within a website.
<b></b> The bold tag will make the text in between the start and end tags bold
<i></i> The italics tag will make the text in between the start and end tags italicised
<u></u> The underline tag will make the text in between
the start and end tags underlined
<div></div> The div tag is ideal for creating sections to be affected by an external 'force' such as CSS or JavaScript. It would be used like so:
<div class="class-name">This text will be affected by that class.</div>
<span></span> The span tag is used a lot like a div tag except, it's usually used just for CSS purposes.
