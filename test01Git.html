Git Reference

List of Helpfull Git and Terminal Commands

General Git Commands

Command Name Description
git --version This will give back the current version of Git
git init This will initiate an empty git repository. Before entering this
command, cd into the folder that you want to initialize.
git add filename This will add the file to the staging area. To select all of the files in a folder, replace filename with "."
git diff This will show what modifications were made between the current staging and the last one.
git diff --staged
git commit -a -m'insert commit message here' This will commit all of the files in your staging area. Be sure to make the
commit message descriptive.
git status This will show you the files ready to be commited

Branch Specific Commands

Command Name Description
git branch Will list all of the branches in the entire repo
git branch branchName Will create a new branch
git checkout branchName Will switch to the specified branch
git checkout -b branchName Will create a new branch and immediately switch over to it. This will help to
eliminate errors by letting you just keep going rather then having to switch over to the branch that you just created
git merge branchName Will merge that branch with the one that you're currently in
git branch -d branchName Will delete that branch if it has already been merged
git branch -D branchName Will force a delete even if it hasn't been merged

Helpful Terminal Commands

Command Name Description
pwd Show my Present Working Directory
cd Change Directory
ls List directories in this current directory
mkdir MaKe DIRectory
~ Will take you to the main directory (cd ~)
clear Will clear the terminal window
../ Will take you up a directory (cd ../)
